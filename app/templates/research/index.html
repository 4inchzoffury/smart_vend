Research Tasks

Research Tasks — Track all pre-launch research and ongoing business tasks on a Kanban board.
  • Tasks are grouped into sections (e.g. Legal, Permits, Equipment). Add sections with Add Section.
  • Each task has a status: Done In Progress Blocked Not Started.
  • Set prerequisites on a task to track dependencies — blocked tasks show their blockers.
  • Drag the handle to reorder tasks within a section. Use arrows to reorder sections.
  • Filter by status or section using the filter bar above the board.
