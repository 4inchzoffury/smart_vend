Import Prices
Bulk-load this supplier's price list into Products + ProductSources.
  • CSV / text — paste a comma-separated table with a header row. Recognized columns (any order, any of these spellings): sku, name, brand, category, case_pack_qty (or pack, units/case), case_price (or wholesale), unit_cost (or each), unit_size, min_order, notes.
  • AI extract — paste raw catalog text (PDF copy-paste, email body, screenshot OCR). Claude Haiku will pull rows out. Requires ANTHROPIC_API_KEY.
  • Existing Products match by SKU first, then case-insensitive name (within category). Existing ProductSources for this supplier get refreshed in place — re-imports won't duplicate.
  • The origin on each ProductSource is tagged {{ supplier.name | lower | replace(" ", "_") }}_import so you can spot AI-loaded rows on the product detail page.
