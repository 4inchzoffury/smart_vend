What this does

Claude will search manufacturer and reseller websites (VendGuys, GeniusVend, store.cantaloupe.com, etc.) for updated specs, pricing, warranty terms, and product images for all {{ unit_count }} equipment units. Results are saved back to the database.

  • Uses your Anthropic API key (Claude API tokens apply) and your chosen web search provider
  • Runs in the background — you can navigate away at any time
  • Takes 2–5 minutes depending on how many searches are needed
  • Fields already filled in will only be overwritten if newer data is found

Frequently asked

Does navigating away stop the refresh?
No — once started, the job runs server-side. You can close this tab, go to another page, or even close the browser. Come back to this page to check on it.

How often should I refresh?
Manufacturer pricing changes infrequently. Once a month is usually sufficient, or before any important client meeting.

Can I stop a running refresh?
Not from the UI — you would need to restart the server. The job typically completes in under 5 minutes.
